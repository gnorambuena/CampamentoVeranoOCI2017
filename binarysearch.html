Búsqueda Binaria

Una vez que vean el video, podrán entender este (pseudo)código de BB que en general debiera ser suficiente para (casi) todo lo que necesiten.

// B: fin del espacio de búsqueda

// Requiere que la propiedad sea "monótona": es falsa hasta cierto punto, desde donde es siempre verdadera.

// Busca el primer valor entre A y B que hace la propiedad

    (A < B){
  2. mid = A + (B - A)/2;

  3. if

    ( /* propiedad es cierta en mid */) {
  4. B = mid;

  5. else

  6. A = mid + 1;
  7. }

  8. if

    ( /* propiedad es falsa en A */)
  9. /* quejarse! la propiedad es falsa para todos los valores entre A y B */

  10. else
